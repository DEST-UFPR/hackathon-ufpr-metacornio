# Metacornio - Hackathon de Dados da UFPR
### Página web integrada com Power BI para visualização dos dados da CPA

[Acesso a página](https://hackathoncpa.metacornio.com.br/)

## Solução Proposta: Foco na Acessibilidade

Nossa equipe desenvolveu uma solução com foco na **acessibilidade** e **manutenção simplificada**, garantindo que os usuários tenham **livre exploração dos dados** no navegador.

A solução se concentra em:
1. **Administração 100% Web:** A **Área Administrativa** permite que usuários autorizados façam o *upload* de arquivos de pesquisa para um banco de dados **[MySQL]** **diretamente pelo navegador**, garantindo uma gestão de dados simples e **independente de sistema operacional**.
2. **Visualização Universal:** Os dados são transformados em **Dashboards** (criados com **Power BI**), que são **embutidos** na **Página Web pública**. Isso garante que a comunidade da UFPR acesse as análises da pesquisa **apenas com um navegador**, eliminando a necessidade de *softwares* adicionais.
